## 题目
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

示例 1：
```
输入： 2
输出： 2
解释： 有两种方法可以爬到楼顶。
1.  1 阶 + 1 阶
2.  2 阶
```
示例 2：
```
输入： 3
输出： 3
解释： 有三种方法可以爬到楼顶。
1.  1 阶 + 1 阶 + 1 阶
2.  1 阶 + 2 阶
3.  2 阶 + 1 阶
```

## 解决方法

第`n`层楼梯的方法为从第`n-1`层爬`1`个台阶和从第`n - 2`爬`2`个台阶。

递归很简单，但实测超时。因此，此题只能用遍历。

```js
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
    let i = 1;
    let j = 1;
    for(let k = 1; k < n; ++k) {
        [i, j] = [j, i + j];
    }
    return j;
};
```
